How this was created

  1. Download the map from the official website
  2. 2D Map
    1. Download uNmINeD
    2. Open the map in uNmINeD
    3. Export the map
  3. Game Maps
    1. Download Papyri & install dependencies
    2. Run Papyri: python .\papyri.py --world ">Drehmal Map Path<" --type java --output "./GameMaps"
  4. 3D Map
    1. Download Bluemap CLI (or use the practical Docker image)
    2. Create configs for all the worlds
    3. Run the CLI / Container
  5. Isometric Map
    1. Download Minecraft Overviewer & install dependencies (or use the practical Docker image I made)
    2. Download & edit my config
    3. Run the render
    4. The --genpoi of Overviewer broke with 1.17, so instead use the alternative I wrote: POIGen
    5. Download Minecraft 1.18.2 with Fabric and install Rend.
    6. Start the instance and load the Drehmal datapack.
    7. Load into a world.
    8. Open the Rend Interface. Render Blocks. Enter minecraft as namespace. You can render it at 16x16 or resize later. Check the option to change the filename to identifier.
    9. Goto the folder Rend created and rename it to icons. Move it into the Overviewer Render export folder.
    10. Remove the minecraft_ from all the filenames in the icons folder. I suggest using something like RegexRenamer.
    11. Fetch the image of all horse variants from the Minecraft wiki (or anywhere else). Resize to 216x240, convert them to .webp, name them after the internal id (first should be 0 last should be 1030) and put them into the horses subfolder.
    12. Fetch marker images, either by creating them or also pulling them from the Wiki (I like to use mob faces). Resize them to 28x28 and put them into the markers subfolder. (You need the following marker images: armor_stand.png, barrel.png, bee.png, book.png, cat.png, chest.png, chest_red.png, chicken.png, cow.png, donkey.png, drowned.png, elder_guardian.png, enchanted_book.png, fox.png, horse.png, item_frame.png, lectern.png, llama.png, map.png, marker_tower_red.png, marker_town.png, mooshroom.png, ocelot.png, panda.png, parrot.png, pig.png, polar_bear.png, rabbit.png, sheep.png, spawner.png, turtle.png, villager.png, wolf.png)
  6. Insert all folders into the page folder, you can get it from Github
    7. , you probably need to rename the folders to 2d, 3d, iso & game.
  7. Optional: Optimize all PNGs to save storage/traffic, e.g. with oxipng.
  8. Put the whole thing on a server with enough storage.
  9. Expose the whole thing with a webserver like Static Web Server & maybe put it behind Cloudflare with a Cloudflare Tunnel
