How this was created

  1. Download the map from the official website (https://www.drehmal.net/)
  2. 2D Map
    1. Download uNmINeD (https://unmined.net/)
    2. Open the map in uNmINeD
    3. Export the map
  3. Game Maps
    1. Download Papyri (https://github.com/jason-green-io/papyri)
    2. Run Papyri: python .\papyri.py --world ">Drehmal Map Path<" --type java --output "./GameMaps"
  4. 3D Map
    1. Download Bluemap CLI (or use the practical Docker container with Docker Compose) (https://bluemap.bluecolored.de/wiki/getting-started/Installation.html#using-bluemap-on-the-cli--standalone)
    2. Create configs for all the worlds
    3. Run the CLI / Container
